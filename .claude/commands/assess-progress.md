# Assess Progress

Invoke the product-manager agent to review the current codebase and update the MVP Progress Log.

## Tasks
1. Inspect existing backend and frontend structure
2. Compare the current system against the MVP Definition of Done (see `.claude/rules/mvp-delivery.md`)
3. Identify completed, partial, and missing pieces
4. Update `.claude/docs/mvp-progress.md` with findings
5. Recommend the next highest-priority task

## Constraints
- Do not start any implementation
- Assessment only — read and report
- Follow output format in `.claude/rules/collaboration.md`
